# Wicked-Sales
A full stack e-commerce web application

## Languages Used

- React
- JavaScript
- Node.js
- Postgresql

## Live Demo

Link: https://wicked-sales.blakeros.com/

## Features

- View product details
- Filter products by category
- Add products to cart
- Review and checkout products from cart
